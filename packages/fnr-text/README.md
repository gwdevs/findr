<h2 align="left">Installation</h2>

**@findr/text** is available as a npm package📚

- npm:

  ```
  npm install @findr/text
  ```

- yarn:

  ```
  yarn add @findr/text
  ```

  - pnpm:

  ```
  pnpm add @findr/text
  ```
