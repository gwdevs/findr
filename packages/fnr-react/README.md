<h2 align="left">Installation</h2>

**@findr/react** is available as a npm package📚

- npm:

  ```
  npm install @findr/react
  ```

- yarn:

  ```
  yarn add @findr/react
  ```

  - pnpm:

  ```
  pnpm add @findr/react
  ```
